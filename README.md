# autoFabrication

Монорепозиторий для локальной производственной системы.

## Структура

- `src/client` - Next.js 16 + TypeScript + VKUI + SCSS Modules + Zustand
- `src/server` - Node.js + Express + TypeScript + Prisma

## Команды

```bash
npm install
npm run dev
npm run build
npm run lint
```

`npm run dev` запускает клиент и сервер одновременно с обновлением в реальном времени.

## Следующий шаг

Следующим этапом можно добавлять Prisma-модели `Role` и `User`, REST API и сид администратора.
